503 Vercel edge handler error

Vercel cannot handle edge requests — site is offline

The server attempted to process an edge function request but failed. The site is temporarily offline for edge requests. This page indicates the deployment couldn't serve the request from an edge runtime.

What happened
  • Edge runtime did not respond or is misconfigured.
  • The request could not be routed to a fallback server.
  • Deployment may be paused or contains an incompatible edge handler.
Retry Check status

If you are the site owner: check your Vercel dashboard, verify your edge function configuration, and confirm there are healthy regions for edge runtime.

Quick troubleshooting
  1. Open the Vercel deployment dashboard.
  2. Check edge function logs and recent failures.
  3. Temporarily disable edge middleware to serve from serverless functions.
  4. Verify runtime/Node version and environment variables.
Reference code: HTTP 503 — Service Unavailable
If you need help, contact the site administrator or try again in a few minutes.
